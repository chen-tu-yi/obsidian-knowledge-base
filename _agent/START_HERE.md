## Vault Purpose

這個 Obsidian vault 是我的長期知識庫，用來沉澱：

- Quant Finance
- Trading Strategy
- Resume and Career
- Software Projects
- Agent Workflow
- Personal Knowledge System

---

## Main Workflow

新資料處理流程：

1. 放入 `00_Inbox/`
2. 使用 Copilot Command：`KB - Process Active Note`
3. 產出 Literature Note 草稿
4. 拆成 Atomic Notes
5. 加入對應 MOC
6. 人工確認後改成 `status: evergreen`

---

## Key Folders

- `00_Inbox/`：尚未整理資料
- `01_Sources/`：來源資料
- `02_Literature Notes/`：忠於來源的整理
- `03_Atomic Notes/`：長期可複用概念
- `04_MOC/`：主題地圖
- `05_Projects/`：專案知識
- `06_Prompts/`：Agent prompts
- `07_Index/`：tag 與索引
