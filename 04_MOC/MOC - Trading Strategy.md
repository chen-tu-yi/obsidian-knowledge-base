# MOC - Trading Strategy

## Purpose

此 MOC 用來整理交易策略研究中的來源可信度、核心概念、方法框架、風控限制與後續待驗證問題。

## How to Use This MOC

建議閱讀順序：

1. 先讀 [[交易策略來源可信度分級]]，決定哪些策略值得研究。
2. 再讀 [[不對稱交易機會]] 與 [[R Multiple 交易評估]]，理解策略評估尺度。
3. 最後用 [[交易 Playbook]] 把候選 setup 轉成可測試流程。
