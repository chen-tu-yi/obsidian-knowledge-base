## Indexing Principle

Tags are filters. MOCs and project index files are the primary navigation layer.

When adding new material:

1. Put raw material in `00_Inbox/`.
2. Create a source or literature note.
3. Extract reusable atomic notes only when the idea is durable.
4. Link to the relevant MOC or project index.
5. Keep status as `needs_review` until manually confirmed.
